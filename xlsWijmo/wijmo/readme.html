Welcome to Wijmo 5 Enterprise, a new generation of JavaScript controls.

This is an Evaluation version of Wijmo 5. To buy your copy now, please visit the Wijmo Purchase page, or to check the latest updates, visit the Wijmo 5 page.

This zip file contains three folders:

  • Dist contains Wijmo 5 distributable files, including the Wijmo controls, CSS styles, and directives for use in AngularJS applications.
  • Samples contains sample applications that show the Wijmo 5 controls in action.
  • VsTemplates contains Visual Studio project templates for Wijmo 5 applications.

To get started, check out the samples and see how to use Wijmo 5 in your applications. Happy coding!

If you need additional help, check out the Wijmo 5 Documentation.
